Install GIF Perfect on Windows

The Windows installer is currently unsigned — code-signing certificates are pending. Until they land, Windows will show a SmartScreen warning that you can dismiss in two clicks. The whole setup takes under a minute.

Why does Windows show a warning? Microsoft's SmartScreen filter flags installers that haven't yet built up reputation with a paid EV certificate. The app itself is safe — it's the same code shipped in the macOS build that's signed and notarized by Apple. EV signing for Windows is on the roadmap.

The 4 steps

1

Download the installer

Click the Windows download button below. The file is GIF Perfect Setup .exe (~110 MB, NSIS installer).

⬇ Download for Windows

Save it to Downloads or anywhere convenient.

2

Click through SmartScreen

Double-click the downloaded .exe. Microsoft Defender SmartScreen may appear with a blue dialog:

"Microsoft Defender SmartScreen prevented an unrecognized app from starting. Running this app might put your PC at risk."
  1. Click More info
  2. Click Run anyway

If the dialog doesn't appear at all, skip to step 3 — you're done with this part.

3

Run the setup wizard

The standard NSIS installer opens:

  • Choose an install location (the default is fine)
  • Click Install
  • Wait ~15 seconds — done
4

Launch and activate

Open GIF Perfect from the Start menu. The free tier (3 conversions/day, with watermark) works immediately — no sign-up.

If you bought a Creator, Studio, or Studio Batch licence, paste your licence key in Settings → Activate Licence to remove the watermark and unlock unlimited conversions.

Verify the install worked

  1. Open Start menu → GIF Perfect
  2. Drop a video onto the app window — or click to browse
  3. Pick a target size (15 MB / 25 MB / 99 MB / Custom) and click Generate
  4. Your output GIF chunks land in the same folder as the source video

Common errors

"Windows protected your PC" — no Run anyway button visible

Click the small More info link first. The Run anyway button only appears after that link is clicked.

"This app can't run on your PC"

Check your Windows version: Windows 10 (64-bit) or Windows 11 required. Run winver in the Start menu to check. 32-bit Windows is not supported.

Antivirus quarantines the installer

Some corporate antivirus (Norton, McAfee, Cylance) deep-scans unsigned installers. If it's blocked outright, add the file as an exception or wait for the EV-signed release. SHA-256 checksums for every release are published on the GitHub releases page.

App opens but FFmpeg isn't found

FFmpeg ships bundled with the installer. If you see a "FFmpeg not found" error, your antivirus likely deleted the bundled binary post-install. Re-install with antivirus paused, or add the GIF Perfect install folder as an exception.
