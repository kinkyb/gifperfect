## App Features
- **GIF Chunks mode**: test segment ‚Üí estimate MB/s ‚Üí calculate chunk duration ‚Üí split video into GIF chunks at target size
- **Frames Only mode**: standalone JPG frame extraction without GIF generation
- **Watermark**: Free tier ‚Äî full-image 9-position tiled `drawtext` overlay via FFmpeg
- **Target size presets**: 15 MB (X/Twitter) / 25 MB (Discord) / 99 MB (OF) / custom
- **Resolution**: 480p / 640p / 1080p / original
- **FPS**: 15 / 24 / 30
- **Frame extraction**: optional, saves JPGs alongside GIFs every N seconds

## FFmpeg
- Dev: resolved via `static_ffmpeg.add_paths()` + `shutil.which('ffmpeg')`
- Packaged: static-ffmpeg binaries bundled by PyInstaller spec into `.app`/`.exe`

## Licence Key Format
- `GIFP-XXXX-XXXX-XXXX` ‚Äî Creator / Studio (standard, no batch mode)
- `GIFB-XXXX-XXXX-XXXX` ‚Äî Studio Batch (unlocks batch mode via prefix detection)
Generated by webhook on Acaption API server. Keys stored in `/opt/acaption-api/gifperfect_keys.txt` (append-only log).
Licence file on client: `~/.gifperfect_licence` ‚Äî JSON `{"key": "...", "batch": true/false}`.

## Resend DNS (gifperfect.com) ‚Äî Complete
All records added to Namecheap Advanced DNS:
- ‚úÖ DKIM TXT: `resend._domainkey`
- ‚úÖ TXT: `send` ‚Üí `v=spf1 include:amazonses.com ~all`
- ‚úÖ TXT: `_dmarc` ‚Üí `v=DMARC1; p=none;`
- ‚ùå MX: `send` ‚Üí `feedback-smtp.eu-west-1.amazonses.com` ‚Äî Namecheap Host Records editor has no MX type; subdomain MX not supported. Skip ‚Äî only affects bounce feedback, not deliverability.
